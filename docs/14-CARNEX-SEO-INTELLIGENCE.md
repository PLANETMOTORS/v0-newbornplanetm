## TECHNICAL SEO: EV INVENTORY FILTERS

### Dynamic URL Structure for EV Keywords

```
/inventory?fuelType=Electric                     → "electric cars"
/inventory?fuelType=Electric&make=Tesla          → "used Tesla"
/inventory?fuelType=Electric&make=Tesla&model=Model+Y  → "Tesla Model Y"
/inventory?fuelType=Electric&maxPrice=25000      → "EV under $25k"
/inventory?fuelType=Electric&maxPrice=40000      → "EV under $40k"
/inventory?fuelType=Electric&range=400           → "400km range EV"
/inventory?fuelType=Electric&drivetrain=AWD      → "all-wheel drive EV"
```

### Schema Markup for EV Pages

```json
{
  "@type": "Vehicle",
  "vehicleEngine": {
    "@type": "EngineSpecification",
    "fuelType": "Electric"
  },
  "driveWheelConfiguration": "AllWheelDriveConfiguration",
  "fuelEfficiency": "0L/100km (Electric)",
  "additionalProperty": [
    {
      "@type": "PropertyValue",
      "name": "Battery Range",
      "value": "450 km"
    },
    {
      "@type": "PropertyValue", 
      "name": "Battery Health",
      "value": "94%"
    }
  ]
}
```
