## Technical SEO Checklist

### Before Launch

- [ ] Submit sitemap to Google Search Console
- [ ] Submit sitemap to Bing Webmaster Tools
- [ ] Verify Google Business Profile
- [ ] Set up Google Analytics 4
- [ ] Set up Google Tag Manager
- [ ] Configure Meta Pixel
- [ ] Test all structured data in Rich Results Test
- [ ] Verify robots.txt is correct
- [ ] Check Core Web Vitals

### After Launch (Week 1)

- [ ] Request indexing for key pages
- [ ] Set up rank tracking (Ahrefs, SEMrush, or free alternative)
- [ ] Monitor Search Console for crawl errors
- [ ] Check mobile usability report
- [ ] Submit to Bing Places for Business

### Monthly Maintenance

- [ ] Review Search Console performance
- [ ] Check for new crawl errors
- [ ] Update sitemap with new inventory
- [ ] Monitor backlink profile
- [ ] Publish 2-4 blog posts
- [ ] Outreach to 10-20 link prospects
